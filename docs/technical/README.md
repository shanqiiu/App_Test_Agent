## 使用指南

### 新成员入门
1. 先阅读 [术语表](./术语表.md) 了解基本概念
2. 再看 [技术栈与工具](./技术栈与工具.md) 了解技术选型
3. 参考 [调研文档](../research/) 了解研究背景

### 技术调研
1. 在 [技术栈与工具](./技术栈与工具.md) 查找相关工具
2. 查阅 [学术研究资源](../references/学术研究.md) 了解理论基础
3. 参考 [开源项目资源](../references/开源项目.md) 寻找实现案例

### 方案设计
1. 查阅 [术语表](./术语表.md) 明确概念定义
2. 参考 [技术栈与工具](./技术栈与工具.md) 选择技术
3. 查看 [研究路线图](../planning/研究路线图.md) 了解整体规划
