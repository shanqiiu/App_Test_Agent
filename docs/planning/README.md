## 使用指南

### 规划新任务
1. 查看 [研究路线图](./研究路线图.md) 确定当前阶段
2. 参考 [待研究问题](./待研究问题.md) 选择研究方向
3. 确定优先级和时间安排
4. 记录到相应的任务清单

### 跟踪进度
1. 定期更新 [研究路线图](./研究路线图.md) 的完成状态
2. 在 [待研究问题](./待研究问题.md) 中标记问题状态
3. 记录遇到的新问题和发现

### 调整规划
1. 根据实际进展调整时间线
2. 识别新的风险和挑战
3. 更新优先级排序
